# CoffeCord

O CoffeCord é um fork comunitário do **Equicord** (GPL-3.0-or-later), criado para manter o projeto vivo e testar ideias novas. A base continua sendo Equicord e Vencord — obrigado às equipes e contribuidores desses projetos.

## Guia rápido
- Requisitos: Node 18+ e pnpm (versão indicada em `package.json`).
- Instalar dependências: `pnpm install`
- Build desktop: `pnpm build`
- Build web: `pnpm buildWeb`
- Injetar/reparar no Discord: `pnpm inject` (ou `pnpm repair` / `pnpm uninject`)
- Desenvolvimento: `pnpm dev` (watch + rebuild)

## Observações
- Repositório oficial: `https://github.com/CoffeCord/CoffeCord`.
- Licença: GPL-3.0-or-later. Por ser fork do Equicord (também GPL), forks descendentes devem manter essa licença.
- Ao abrir issues, inclua plataforma, tipo de build e passos para reproduzir.

## Créditos
- Equicord: equipe e contribuidores pelo código e recursos originais.
- Vencord/Vesktop: autores e comunidade do ecossistema de client mods.
